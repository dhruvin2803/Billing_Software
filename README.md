# Billing_Software

The Project Titled "Billing Software" is a Record keeping system. It is a simple database project In C++ Language. It is done using file handling in C++. The Record of restaurant can be added, updated, searched and deleted. It is a simple project made using console application of C++. This means no graphics components are added.

In This Project “Billing Software” details can be added along with their Item Code, Name of Product and Price of Product as and when required. The search can be done using item code of the product. The product can also be deleted using Item Code. As Well As modification is possible.

## You need to install the following libraries before running this code:- 
1.  #include<iomanip.h> - This Header file is used to Utilize input/output manipulation function.  Example:-  setw(), setprecision().
2.	#include<conio.h> - This Header file is used to Utilize The console input/output and miscellaneous functions. Example:- getch(),getche().
3.	#include<string.h> - This Header file is used to Utilize string handling functions. Example:- strcpy(),strcmpi().
4.	#include<stdio.h> - This Header file is used to Utilize The Cursor Movement function.Example:- gotoxy().
5.	#include<fstream.h> - This Header file is used to Utilize The Data Filer Handling Management Functions. Example:- open(),close(),read(),write().
